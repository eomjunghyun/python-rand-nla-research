# 비균일 HSBM 스케일링 실험

이 폴더는 초기 비균일 HSBM 스케일링 실험을 위해 만들었던 legacy 작업 폴더다. 현재 [src/common.py](/Users/eomjeonghyeon/Documents/github_project/python-rand-nla-research/src/common.py)는 균일 HSBM API만 유지하므로, 이 폴더는 실행 대상 실험 폴더가 아니라 과거 구조를 설명하는 기록으로 둔다.

## 현재 상태

이 폴더에는 실행 대상 노트북을 두지 않는다. 새 하이퍼그래프 실험은 `experiments/균일 HSBM 실험/`을 기준으로 관리한다.

## 작성 규칙

앞으로 이 폴더에 문서를 추가할 경우 기본 설명은 한글로 작성한다. 필요한 실험 용어와 함수명은 영어를 섞어 쓸 수 있다.
