# 비균일 HSBM 스케일링 실험

이 폴더는 초기 비균일 HSBM 스케일링 실험을 위해 만들었던 작업 폴더다. 현재 실제 실험 노트북은 `experiments/메인 실험/` 아래의 식별자 기반 파일명으로 정리되어 있다.

## 현재 상태

이 폴더에는 실행 대상 노트북을 두지 않는다. 비균일 HSBM의 `n` 변화, 큰 `n` 변화, `K` 변화 실험은 모두 `메인 실험` 폴더를 기준으로 관리한다.

## 작성 규칙

앞으로 이 폴더에 문서를 추가할 경우 기본 설명은 한글로 작성한다. 필요한 실험 용어와 함수명은 영어를 섞어 쓸 수 있다.
